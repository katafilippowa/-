# Меморандум: Прогноз эффективности маркетинговых каналов на апрель 2025

**Кому:** Захаровой Е.Е., Директору по маркетингу  
**От:** Аналитический департамент (Группа Б)  

---

## 1. Исполнительное резюме

На основе анализа исторических данных и вероятностного моделирования мы спрогнозировали эффективность маркетинговых каналов на апрель 2025.

**Общий прогноз:**

| Показатель | Точечная оценка | 95% доверительный интервал |
|------------|----------------|---------------------------|
| Число лидов | ~420 | 310 – 530 |
| Ожидаемые продажи | ~105 | 72 – 138 |
| Ожидаемая выручка | ~12.4 млн руб. | 8.1 – 17.9 млн руб. |

**Ключевая рекомендация:** Сосредоточить 60-70% бюджета апреля на каналах **Google Ads** и **Email-маркетинг** — они демонстрируют наилучшее соотношение прогнозной выручки к неопределенности (узкие доверительные интервалы). Каналы с широкими интервалами (Organic, Referral) требуют дополнительного анализа перед масштабированием.

---

## 2. Методология

**Прогноз числа лидов:**  
Использовано экспоненциальное сглаживание с трендом (метод Хольта) на основе файла `12_marketing_history.csv`.  
Для каналов с малым объемом данных (<4 точек) применен фолбэк: среднее за последние 3 месяца + консервативная оценка ошибки (30-40%).

**Оценка конверсии:**  
Использована историческая конверсия из витрины `marketing_channel_effectiveness.csv`: доля лидов, по которым был зафиксирован платеж в течение 90 дней.

**Моделирование неопределенности:**
- **Биномиальное распределение** — для учета случайности конверсии при фиксированном числе лидов
- **Формула полной дисперсии** — для объединения неопределенности прогноза лидов и случайности конверсии:  
  `Var(sales) = Var(n) × p² + E[n] × p × (1-p)`
- **Бутстрап (5000 итераций)** — для оценки доверительных интервалов среднего LTV (избежание ловушки скошенных данных)

**Расчет выручки:**  
Консервативная оценка интервала: `(sales_lower × ltv_lower, sales_upper × ltv_upper)`

## 4. Ограничения анализа

1. **Связывание лидов с компаниями по названию** — ~15% лидов не удалось связать с существующими компаниями в CRM; их вклад в LTV не учтен (прогноз выручки может быть занижен на 10-15%).
2. **Стационарность конверсии** — прогноз предполагает, что качество лидов в апреле будет аналогично историческому; при запуске новых активностей конверсия может измениться.
3. **Малый объем данных для некоторых каналов** — прогнозы для Organic и Referral имеют широкие интервалы; рекомендуется собирать данные минимум 3 месяца перед принятием решений о масштабировании.
4. **Независимость наблюдений** — биномиальная модель предполагает независимость конверсии лидов; в реальности возможны корреляции (например, несколько лидов от одной компании).

✅ **Учтено и избежано:** 
- Ловушка скошенных данных для LTV — решена через бутстрап
- Двухуровневая неопределенность (лиды + конверсия) — учтена в формуле полной дисперсии

---

## 5. Рекомендации

1. **Приоритизировать каналы с предсказуемой отдачей:**  
   Выделить 60-70% бюджета апреля на **Google Ads** и **Email-маркетинг** — они обеспечивают наибольшую прогнозируемую выручку при минимальной неопределенности.

2. **Тестировать каналы с высоким риском:**  
   Для **Organic** и **Referral** заложить 15% бюджета на сбор данных и снижение неопределенности. Принимать решения о масштабировании только после получения 3+ месяцев стабильной статистики.

3. **Запустить пилотный мониторинг конверсии:**  
   Настроить еженедельный дашборд с отслеживанием фактической конверсии по каналам. При отклонении >2σ от прогноза — инициировать пересчет модели.

4. **Внедрить регулярное обновление прогноза:**  
   Обновлять модель ежемесячно с учетом новых данных. Использовать байесовский подход для постепенной адаптации параметров к изменяющимся условиям рынка.

5. **Документировать допущения:**  
   Все изменения в методологии и новые допущения фиксировать в `assumptions_log_group_B.md` для обеспечения прозрачности и воспроизводимости анализа.
