# Анализ конфликтов определений метрик (Группа Б)
## Анализ эффективности маркетинговых кампаний


---

## Введение

В ходе аудита данных выявлено, что ключевые метрики для анализа эффективности маркетинговых каналов по-разному определяются в различных источниках и отделах. Это приводит к противоречивым отчетам и сложностям в принятии решений.

**Цель анализа:** Выявить не менее трех метрик с конфликтующими определениями и предложить единые корпоративные определения для глоссария.

**Метрики для анализа (согласно заданию для Группы Б):**
1. LTV (пожизненная ценность клиента) — упрощенно
2. Конверсия в продажу
3. Время реакции поддержки

---

## МЕТРИКА 1: LTV (Lifetime Value) — упрощенно

### Шаг 1. Сбор определений из разных источников

| Источник | Определение | Формула / Алгоритм | Откуда взято |
|----------|-------------|-------------------|--------------|
| **Финансы (1С)** | Сумма всех платежей клиента за все время | `LTV_fin = SUM(amount) WHERE company_id = X AND amount > 0` | 05_finance_payments.csv, поле amount |
| **Маркетинг** | Прогнозная ценность на основе ARPU и оттока | `LTV_mkt = ARPU × (1 / Churn_Rate)` | Методика расчета эффективности каналов |
| **Аналитика (упрощенная)** | Сумма платежей за фиксированный период (12 мес) | `LTV_12m = SUM(amount) WHERE doc_date >= CURRENT_DATE - 365` | Практика когортного анализа |
| **Продукт** | Сумма платежей минус стоимость поддержки | `LTV_prod = SUM(amount) - SUM(support_cost)` | Учет операционных затрат |
