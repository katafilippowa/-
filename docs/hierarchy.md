# Иерархическая модель корпорации "Образование Будущего" (Группа Б)

## Юридическая структура
- **Холдинг "Образование Будущего"**
    - **ООО "Учись дома"** (B2C-сегмент)
        - Направление: онлайн-курсы для физических лиц
        - Руководитель: Соколова Мария Ивановна
    - **АО "Профи-тренинг"** (B2B-сегмент)
        - Направление: корпоративное обучение
        - Руководитель: Козлов Дмитрий Викторович
    - **ООО "ЛМС-Технологии"** (Платформа)
        - Направление: разработка и поддержка LMS
        - Руководитель: Новиков Александр Александрович

## Функциональная структура (бизнес-процессы)
### 🎯 Домен «Маркетинг и Привлечение»
- Руководитель: Захарова Елена Евгеньевна
- **Бизнес-процесс:** Генерация лидов → Запуск кампаний → Трекинг каналов → Атрибуция CPA/CAC
- **Системы-генераторы:** Marketing Automation, Веб-трекинг
- **Системы-потребители:** CRM (передача лидов), Аналитика (оценка каналов)
- **Ключевые связи:** `lead_id`, `email`, `campaign_id`, `channel_id`

### 👥 Домен «Клиенты и Продажи (CRM)»
- Руководитель: Козлов Дмитрий Викторович
- **Бизнес-процесс:** Регистрация компании → Квалификация лида → Сделки → Онбординг → Аккаунт-менеджмент
- **Системы-генераторы:** CRM-система
- **Системы-потребители:** Финансы (выставление счетов), Поддержка (карточка клиента), Маркетинг (конверсия)
- **Ключевые связи:** `company_id`, `lead_id`, `manager_id`, `status`

### 💰 Домен «Финансы и Платежи»
- Руководитель: Григорьев Геннадий Геннадьевич
- **Бизнес-процесс:** Выставление счетов → Приём оплат → Возвраты → Сверка с CRM
- **Системы-генераторы:** 1С, Банковские шлюзы
- **Системы-потребители:** Аналитика (LTV, ARPU), CRM (статус оплаты)
- **Ключевые связи:** `company_id`, `transaction_id`, `doc_id`

### 🛠️ Домен «Поддержка и Клиентский сервис»
- Руководитель: Волкова Ирина Игоревна
- **Бизнес-процесс:** Регистрация обращения → SLA → Решение → Оценка CSAT
- **Системы-генераторы:** Zendesk, Почтовые сервисы
- **Системы-потребители:** Аналитика (нагрузка на канал), Продукт (баги, UX)
- **Ключевые связи:** `company_id`, `customer_email`, `ticket_id`

### 📦 Домен «Продукты и Обучение (LMS)»
- Руководитель: Новиков Александр Александрович
- **Бизнес-процесс:** Каталогизация курсов → Назначение доступа → Активность в LMS → Сертификация → Расчет Churn/Retention
- **Системы-генераторы:** LMS-платформа, Google Sheets (справочники)
- **Системы-потребители:** Маркетинг (популярность продуктов), Финансы (выручка по продуктам)
- **Ключевые связи:** `product_id`, `course_id`, `company_id`, `user_id`

> 🔑 **Принцип построения:** Домены сгруппированы вокруг сквозных бизнес-процессов, а не файлов. Один файл может участвовать в нескольких доменах через разные ключи, обеспечивая отражение полного жизненного цикла клиентского пути.
