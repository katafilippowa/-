# Паспорт данных: 05_finance_payments.csv

### 1. Идентификация:
- **Система:** 1С
- **Тип источника:** Финансовая/операционная
- **Имя файла:** 05_finance_payments.csv

### 2. Назначение:
Хранение информации о всех платежах клиентов, возвратах и финансовых транзакциях для бухгалтерского учета, анализа доходов и расчета метрик (LTV, выручка, отток).

### 3. Владелец:
- **Подразделение:** Финансовый департамент
- **Контактное лицо:** Финансовый директор / Руководитель отдела учета

### 4. Структура (предполагаемая):
- **Количество полей:** ~8-10
- **Перечень полей:**
  - `payment_id` (int) — уникальный идентификатор платежа
  - `company_id` (int) — идентификатор компании-клиента
  - `doc_date` (date) — дата платежа
  - `amount` (float) — сумма платежа
  - `payment_type` (string) — тип платежа
  - `status` (string) — статус платежа
  - `currency` (string) — валюта
  - `invoice_id` (int) — ссылка на счет
- **Количество записей:** ~10,000-50,000

### 5. Временные характеристики:
- **Период данных:** 2019-01-01 по 2024-12-31
- **Частота обновления:** Ежедневно (ночной импорт из 1С)

### 6. Предварительная оценка качества:
- Возможны пропуски в `invoice_id` для мелких платежей
- Возможны дубликаты при повторном импорте
- Отрицательные суммы в `amount` могут означать возвраты

### 7. Связи с другими источниками:
- **Связан с:** 07_marketing_leads.csv, 01_crm_companies.csv
- **По полям:** `company_id`

### 8. Ограничения:
- Конфиденциальная финансовая информация
- Нельзя передавать внешним сторонам без согласования
